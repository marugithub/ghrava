Free APIs

These are free, no-key APIs that enrich your data automatically. Toggle any off if you prefer to skip the network call.

🚗
NHTSA VIN Decoder
Auto-fills Make, Model, Year, Trim when you enter a VIN on the Vehicles form.
vpic.nhtsa.dot.gov · US Government · No key required
🥫
Open Food Facts
Fallback product lookup for grocery and food barcodes when the primary UPC database has no result.
world.openfoodfacts.org · Open database · No key required
💊
OpenFDA Drug Lookup
Look up drug information (generic name, manufacturer, route) when adding a medication.
api.fda.gov · US FDA · No key required
📍
ZIP Code Auto-Fill
Auto-fills City and State when you type a ZIP code in Property and Contact address fields.
api.zippopotam.us · Free · No key required
📚
Google Books
ISBN lookup for book details (title, author, publisher, cover). Requires a free Google API key.
googleapis.com · Free tier · API key required
