Open Food Facts

world.openfoodfacts.org · Open database · No key required

api.fda.gov · US FDA · No key required

api.zippopotam.us · Free · No key required

googleapis.com · Free tier · API key required

Imports / exports
📤 Export All Data
Downloads a single Excel workbook with all your Ghrava data — one sheet per module. Includes an Instructions sheet explaining every column. Use this to back up, review in Excel, or prepare a file for re-import.
⬇ Download Full Export 📋 Download Blank Template
Template = same workbook with headers only, no data — useful for adding new records without downloading your full export.
📑 Sheet Reference
Each sheet in the exported workbook and what the key column means. Use this when filling in a template for import — the key column is how Ghrava matches and updates existing rows.
📥 Import Ghrava Data
⚠ Before importing: Make a backup — go to Settings → Backup & restore → Run Backup Now. Import updates existing rows and adds new ones. It does not delete anything.
Upload the Ghrava export file (or a template you filled in). Only sheets present in the workbook are processed — delete sheets you don't want to change before uploading. Rows with a matching id are updated. Rows without an id are inserted as new.
📂
Drop Excel file here or tap to browse

🏦 Import Bank Statement
Upload a statement from your bank to import transactions into the Finance module. Supported banks: Chase, Bank of America, Navy Federal, Schwab, Vanguard, TSP, Citi, Discover, Wells Fargo, Capital One, USAA — plus generic CSV. Also accepts OFX and QFX formats.
Open Finance → Import Statement →
Bank statements use the Finance module's Import tab — transactions are matched to accounts and auto-categorized using your category rules.
