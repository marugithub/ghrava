Locked agreements

  • Option B — ALL 25 modules' drawers, not partial.
  • Class name stays .gh-drawer. CSS behavior changes only. One file (shared.css).
  • Tall forms: max-height: 90vh + internal scroll. No special-case handling per form.
  • Backdrop click-to-close preserved.
  • Three width tiers: sm 380px, md 520px (default), lg 720px.

Open questions

Mobile (<480px) — full-screen vs centered?
Lean: full-screen, slide-up, rounded top corners. Same component, media-query only.
Animation — fade+scale or slide?
Lean: fade+scale (0.96 → 1.0) over 180ms. Matches center placement.
ESC-to-close — add now or defer?
Lean: add now. One keydown listener in lt-core.js.

Build sequence

  1. Restyle .gh-drawer-overlay and .gh-drawer in shared.css: center, scale animation, max-height 90vh.
  2. Add .gh-drawer--sm / .gh-drawer--lg size variants.
  3. Add ESC-to-close listener in lt-core.js.
  4. Mobile media query @media (max-width: 480px) → full-screen modal.
  5. Visual regression on 5 reps: inventory item, kid edit, hsa receipt, settings family edit, todo add.
  6. Predeploy gate. Package. Ship.
